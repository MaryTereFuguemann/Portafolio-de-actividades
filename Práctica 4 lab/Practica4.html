- Resumen -

En esta práctica, se realizó el diseño de una página web con ayuda de github, en donde con ayuda de distintos comandos se intentó copiar el diseño de la página de Intrauia.

- Introducción -

En la actualidad, la creación de páginas web es fundamental para cualquier negocio o proyecto personal. HTML (HyperText Markup Language) es el lenguaje de marcado estándar utilizado para construir estructuras y contenidos en la web. Con la ayuda de herramientas poderosas como Visual Studio y GitHub, el proceso de desarrollo web se simplifica considerablemente y se vuelve más colaborativo.

Visual Studio, conocido por su robustez y funcionalidad, ofrece un entorno integrado que facilita la escritura de código HTML y otros lenguajes web como CSS y JavaScript. Con su soporte para completado automático, depuración y extensiones, los desarrolladores pueden crear páginas web modernas de manera eficiente y con menos errores.

GitHub, por otro lado, juega un papel crucial en el flujo de trabajo colaborativo y la gestión de versiones. Como plataforma de alojamiento de código basada en Git, permite a los equipos de desarrollo trabajar juntos en proyectos web, realizar seguimiento de cambios y coordinar las actualizaciones de manera ordenada. Esto es especialmente útil para mantener la coherencia y la integridad del código a medida que evoluciona el proyecto.

En conjunto, HTML junto con Visual Studio y GitHub forman un trío poderoso para la creación y gestión de páginas web modernas. Este enfoque no solo facilita el desarrollo individual, sino que también fomenta la colaboración efectiva entre desarrolladores, mejorando la calidad y la eficiencia en la creación de experiencias web dinámicas y atractivas.

Crear una página web con el diseño de una de las páginas de la Universidad Iberoamericana de Puebla, como por ejemplo: IntraUIA, moodle, etc.

- Conclusiones -

Con Visual Studio proporcionando un entorno robusto para la programación y GitHub facilitando la gestión de versiones y el trabajo colaborativo, los desarrolladores pueden concentrarse más en la innovación y menos en las complejidades técnicas. Así, HTML junto con estas herramientas modernas no solo hacen posible la creación de páginas web atractivas y funcionales, sino que también impulsan la productividad y la calidad en el desarrollo web contemporáneo.
