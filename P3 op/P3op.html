Cables de red IP

- Resumen -

En esta práctica, se conocerá, identificará y comprobará el funcionamiento de una red de área local utilizando switches y Router.

- Introducción -

Para identificar y caracterizar el funcionamiento de un switch y un router, es esencial entender que un switch permite la conexión y comunicación entre múltiples dispositivos dentro de una misma red local (LAN), mientras que un router facilita la conexión entre diferentes redes, dirigiendo el tráfico de datos de manera eficiente. Para elaborar una red con asignación automática de IP, se selecciona el tipo de cable adecuado y se utiliza un switch para conectar al menos tres computadoras y una que actúe como servidor. Posteriormente, se comprueba la conectividad utilizando el comando PING y se verifica el acceso a la página web del servidor. De manera similar, se puede configurar una red con asignación automática de IP utilizando un router para conectar al menos tres computadoras, ya sea de manera alámbrica o inalámbrica, y una con servicios de servidor. Nuevamente, se verifica la conectividad con el comando PING y el acceso a la página web del servidor. Finalmente, ambas redes se pueden unir, asegurando la correcta asignación automática de IP y seleccionando el tipo de cable adecuado, para comprobar el intercambio de archivos entre los dispositivos.

- Materiales -

- 6 computadoras

- 1 switch

- 1 router

- Cables de red directos y cruzados

- Desarrollo -

1. Identificar y caracterizar el funcionamiento del switch y el router.

2. Elaborar una red, utilizando la asignación automática de IP, seleccionado el tipo de cable adecuado y un switch conectando mínimo 3 computadoras y una con servicios de servidor.(Comprueba que es posible encontrar dispositivos en la red, puedes usar el comando PING, y comprueba el acceso a la página web del servidor)

3. Elaborar una red, utilizando la asignación automática de IP, seleccionado el tipo de cable adecuado y un router conectando mínimo 3 computadoras (alámbrica o inalámbricamente) y una con servicios de servidor. (Comprueba que es posible encontrar dispositivos en la red, puedes usar el comando PING, y comprueba el acceso a la página web del servidor)

4. Une ambas Redes, utilizando la asignación automática de IP, seleccionado el tipo de cable adecuado y comprueba el intercambio de archivos.

- Resultados -

Tanto los switches como los routers son componentes esenciales en las redes informáticas, pero cada uno cumple un rol específico en la transmisión de datos. Es crucial comprender sus diferencias para una adecuada configuración y gestión de la red.

Funcionamiento de un switch:

La función principal es conectar dispositivos dentro de una misma red local (LAN). Opera en la capa 2 del modelo OSI (Enlace de datos), utilizando direcciones MAC para identificar dispositivos únicos en la red. Su encaminamiento consiste en enviar paquetes de datos al puerto de destino específico según la dirección MAC, y también permite la comunicación simultánea a todos los dispositivos conectados mediante broadcast.

Funcionamiento de un router:

La función principal es conectar diferentes redes entre sí, incluyendo LAN, WAN e Internet. Opera en la capa 3 del modelo OSI (Red), utilizando direcciones IP para identificar dispositivos y redes. Su encaminamiento consiste en enrutar paquetes de datos a la red de destino correcta según la dirección IP y las tablas de enrutamiento. Además, implementa funciones de seguridad como firewall y control de acceso.

Caracterización del funcionamiento del switch y router.

A continuación, se presentan imágenes de la elaboración de las redes con ayuda del switch y router. Así como la asignación automática de IPs. Además, con ayuda del comando PING se comprobó el acceso a la página web del servidor.

- Conclusiones -

En resumen, los dispositivos de red cumplen funciones esenciales para la comunicación y la seguridad en diversas configuraciones de redes. Aquellos que operan en la capa 2 del modelo OSI (Enlace de datos) utilizan direcciones MAC para conectar dispositivos dentro de una misma red local (LAN), enviando paquetes de datos al puerto destino específico y permitiendo comunicaciones simultáneas a través de broadcast. Por otro lado, los dispositivos que trabajan en la capa 3 del modelo OSI (Red) utilizan direcciones IP para conectar diferentes redes, incluyendo LAN, WAN e Internet, enrutan paquetes de datos a la red de destino correcta basándose en direcciones IP y tablas de enrutamiento, y además implementan funciones de seguridad como firewalls y control de acceso. Estas capacidades son fundamentales para la interconexión eficiente y segura de dispositivos y redes.
