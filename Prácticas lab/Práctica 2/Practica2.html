- Resumen -

En esta práctica, se configuró una red local junto con el acceso a recursos de un equipo en red con ayuda de Xaamp.

- Introducción -

XAMPP es un paquete de software gratuito y de código abierto que se utiliza para crear y gestionar servidores web locales. Se trata de una herramienta versátil y práctica para el desarrollo web local que facilita la creación, prueba y depuración de sitios y aplicaciones web.

- Desarrollo -

Identifica la Ip de acceso de la red de tu router y accede a su ruta de configuración.

En tu computadora instala Xampp para configurar un servidor local, habilita los servicios de Apache y monta un sitio web: https://www.apachefriends.org/es/index.html

Habilita el acceso al puerto 80 desde el firewall del equipo configurado como el servidor local.

- Conclusiones -

La creación de una red local utilizando XAMPP es una solución eficiente y accesible para desarrollar, probar y desplegar aplicaciones web en un entorno controlado. XAMPP, al integrar Apache, MySQL, PHP y Perl en un solo paquete fácil de instalar, simplifica considerablemente la configuración de un servidor local. Esto permite a los desarrolladores concentrarse en la creación y el perfeccionamiento de sus proyectos sin preocuparse por las complejidades de la configuración del servidor.
